# Regime-Aware Fixed Income Portfolio Allocation Model

A dynamic fixed-income allocation strategy that adapts to changing geopolitical and economic policy regimes.

## Overview

This model implements a regime-aware asset allocation engine designed for:
- **Primary Mandate:** Capital preservation
- **Secondary Mandate:** Positive risk-adjusted returns

The strategy uses macroeconomic indicators to detect market regimes and adjusts portfolio allocations accordingly.

## Files

- `Regime_Aware_Asset_Allocation_Engine.ipynb` - Main notebook with full implementation and documentation
- `master_indicators_dataset_monthly.csv` - Monthly macroeconomic indicators
- `master_asset_data.csv` - Asset price data
- `master_indicators_dataset_processed.csv` - Processed indicator data
- `Outputs/` - Generated outputs and visualizations

## Usage

Open and run the Jupyter notebook:
```bash
jupyter notebook Regime_Aware_Asset_Allocation_Engine.ipynb
```

All parameters, methodology, and detailed documentation are included in the notebook.
